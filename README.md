DHBW-Dualis-App
===============

Egit in Eclipse einbinden:
http://unicase.blogspot.de/2011/01/egit-tutorial-for-beginners.html
Es muss die http-Addresse verwendet werden.


Bearbeiten einrichten:
 - Neuen Branch anlegen: Rechtsklick->Team->Switch To->New Branch

 In diesem Branch kann dann Problemlos gearbeitet werden.
 Nach dem etwas geändert wurde über Committ kurz beschreiben was geändert wurde.
 Wennn man fertig ist können die Änderungen über Push an den Githubserver übermittelt werden.
 Aktuelle änderungen können über Pull heruntergeladen werden.
